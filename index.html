設定 Google Maps API 金鑰

此地圖使用 Google Maps JavaScript API，
需要先填入你的 API 金鑰才能顯示地圖。

📋 如何取得免費 API 金鑰（約 5 分鐘）

  1. 前往 Google Cloud Console
  2. 登入你的 Google 帳號（黨部 Gmail 即可）
  3. 點左上角「選取專案」→「新增專案」，隨意命名（例如：屏東黨部）
  4. 左側選單：「API 和服務」→「程式庫」
  5. 搜尋 Maps JavaScript API → 點選 → 「啟用」
  6. 左側選單：「API 和服務」→「憑證」→「建立憑證」→「API 金鑰」
  7. 複製產生的金鑰（格式類似 AIzaSy......）貼到下方
  8. 建議在「限制金鑰」中設定 HTTP 參照網址為你的電腦，避免被盜用

✅ 金鑰只存在你的瀏覽器本機，不會上傳任何伺服器
